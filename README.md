## API de Cadastro e Busca de Usuário
Este projeto é uma aplicação backend desenvolvida em PHP para gerenciar o cadastro e a busca de usuários. Ele permite que os usuários sejam cadastrados com informações básicas e depois possam ser recuperados via consultas específicas.

## Tecnologias Utilizadas
O projeto foi desenvolvido utilizando as seguintes tecnologias:

PHP: Linguagem de programação para o backend da API.
MySQL: Banco de dados relacional para persistência de dados de usuários.
Postman: Ferramentas para testar as rotas da API durante o desenvolvimento.
